number-sequence-calculator
==========================

### Calculator Instructions
1. Download entire fileset for the number-sequence-calculator on to your hard drive
2. Execute .html file in browser
3. Follow on-screen instructions

Note: JavaScript must be enabled, and the browser must have a working internet connection

### Unit Test Instructions
1. Open browser developer console (e.g. Firebug, Chrome Developer Tools)
2. Run from console command line: unitTest.init();
3. Results will be displayed beneath standard result area.
